How it works

Add custom domains that point to 127.0.0.1 in your hosts file. This allows you to access your local projects using custom domains like app.test instead of localhost:8000.

Requirements
  • Admin/sudo access to modify hosts file
  • Domain name (e.g., myapp.test)
  • Running project to test
Best Practices
  • Use .test or .local domains
  • Avoid real TLD extensions
  • Keep names simple and memorable
